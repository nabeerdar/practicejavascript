Variables

Var

It is global scoped. Example is shown in conditional statements.
It is the keyword that tells JavaScript you're declaring a variable.

Let

Variables defined with let must be Declared before use.
It is block scoped. Example is shown in conditional statements.
The let keyword was introduced in ES6 (2015). Variables defined with let cannot be Redeclared.

Const

Variables defined with const can not be redeclared.
It is block scoped. Example is shown in conditional statements
const allows you to only modify the value of the array, but the reference to the array cannot be changed.

Operators

+, -, *, **, /, %, ++, --, =, ==, ===

Difference between == and ===

The difference between == and === is that: == converts the variable values to the same type before performing comparison.
This is called type coercion. === does not do any type conversion (coercion) and returns true only if both values and types are identical for the two variables being compared.
